### Square Calculator

This simple C++ program calculates the square of a given number.

#### Usage

1. Run the program.
2. Enter a number when prompted.
3. Receive the square of the entered number.

#### Sample Output

```plaintext
Enter a number : 5
The square of the entered number is : 25
```
